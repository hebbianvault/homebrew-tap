# Hebbian Homebrew Tap

Customer-installable adapter packages for the Hebbian tenant brain, distributed via Homebrew.

## Install

```bash
brew tap hebbianvault/tap
brew install hebbian-mcp
```

## Available formulae

| Formula | What it installs | Source |
|---|---|---|
| `hebbian-mcp` | The MCP adapter that drops the Hebbian tenant brain into Claude Code, Claude Desktop, Cursor, and any MCP-compatible agent. | [`hebbianvault/hebbian-mcp`](https://github.com/hebbianvault/hebbian-mcp) |

## Auto-bumped

Formulae here are auto-bumped on tagged releases of the upstream repos via the action wiring in each source repo's release pipeline.
